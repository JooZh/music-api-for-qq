## 获取MV播放链接

#### 全路径

```
http://localhost:8080/music/api/mv_info
```

#### 前置条件

> 需要获得mv的 mid 
> 获取歌曲相关接口。

#### 参数说明

| 参数名   | 默认值 | 类型   | 必填 | 可选参数                          | 说明               |
| :------- | ------ | ------ | ---- | --------------------------------- | ------------------ |
| mv_mid | v00149ipnk5 | string | *    | 无 | 唯一标识 |


#### 请求方式

参数结构和名称不可改变，只能修改参数值。
该接口目前只支持单个歌曲的数据返回。

```js
axios.get(url, {
  params:{
    mv_mid:"v00149ipnk5"
  }  
}).then((response)=>{
  response.data
})
```
